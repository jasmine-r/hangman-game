Rules & How the Game Works

Objective

Guess the word!

Steps

  • Guess a letter

    • If correct, the dashes which comprise the word will be updated accordingly.

    • If incorrect, the incorrect guess will be one guess closer to you losing. (max 7 incorrect guesses)

Description

The system will randomly select a word. The word to be guessed will be represented by a row of dashes, where each dash represents a letter in the word. The player (you) will be presented with this row of dashes and the player must guess a letter which occurs in the word. If the letter does occur in the word, the letter will replace the dash(es) in the appropriate positions. Be careful! Incorrect guesses for letters are tallied; you lose if you guess a letter incorrectly 7 times.
